Slidex.js

Một thư viện slider/carousel đơn giản và nhẹ được xây dựng bằng JavaScript thuần túy.

Dùng thử

Slidex là một thư viện slider/carousel nhẹ được xây dựng bằng JavaScript thuần túy. (Mã nguồn có sẵn trên GitHub)

  • Không có thư viện phụ thuộc
  • Hoàn toàn tùy biến được với CSS
  • Hỗ trợ tự động chạy (autoplay)
  • API đơn giản và trực quan
  • Gọn nhẹ và được tối ưu hóa

Cài đặt

Phương thức Các bước
Tải trực tiếp Tải file ZIP

Chỉ cần giải nén tệp ZIP và sử dụng các tệp slidex.min.js và slidex.min.css trong dự án của bạn.

Slidex cung cấp một tệp CSS nhỏ mà bạn có thể nhúng trực tiếp qua link CDN. Điều này giúp bạn dễ dàng tích hợp vào dự án và tùy chỉnh khi cần.

Nhúng tệp JavaScript của Slidex vào dự án của bạn:

<script src="https://cdn.jsdelivr.net/gh/sondnpt00343/slidex@v1.0.0/slidex.min.js"></script>
Bạn có thể sử dụng các phiên bản đã phát hành của Slidex tại đây.

Cách sử dụng

Đây là cấu trúc HTML cơ bản bạn cần để thiết lập Slidex:

<div id="my-slider">
    <div>Nội dung slide 1</div>
    <div>Nội dung slide 2</div>
    <div>Nội dung slide 3</div>
</div>

Và đây là cách bạn có thể sử dụng nó với JavaScript:

// Khởi tạo một slider mới
const slider = new Slidex('#my-slider', {
    item: 1, // Số lượng item hiển thị
    loop: true, // Lặp lại slide
    speed: 500, // Tốc độ chuyển slide (ms)
    autoplay: true, // Tự động chạy
    autoplayTimeout: 3000, // Thời gian chờ giữa các lần chuyển slide
    navigation: true, // Hiển thị dấu chấm điều hướng
});

// Chuyển đến slide cụ thể (sử dụng phương thức nội bộ)
slider.moveSlide(1); // Chuyển tới slide tiếp theo

Tùy chọn

Tùy chọn Kiểu dữ liệu Mô tả
item number Số lượng slide hiển thị cùng một lúc (mặc định: 1).
speed number Tốc độ chuyển slide tính bằng mili giây (mặc định: 300).
loop boolean Cho phép lặp lại slider khi đến slide cuối cùng (mặc định: false).
navigation boolean Hiển thị các dấu chấm điều hướng (mặc định: true).
control boolean Hiển thị các nút điều khiển Trước/Sau (mặc định: true).
controlText array Mảng chứa văn bản cho các nút Trước và Sau (mặc định: ['<', '>']).
prevButton string Selector CSS cho nút "Trước" tùy chỉnh (mặc định: null).
nextButton string Selector CSS cho nút "Sau" tùy chỉnh (mặc định: null).
slideBy number | 'page' Số lượng slide sẽ trượt mỗi lần. Nếu là 'page', nó sẽ trượt theo số lượng item (mặc định: 1).
autoplay boolean Kích hoạt chế độ tự động trượt (mặc định: false).
autoplayTimeout number Thời gian chờ giữa các lần trượt tự động (mặc định: 3000).
autoplayHoverPause boolean Tạm dừng tự động trượt khi di chuột qua slider (mặc định: true).

API

Phương thức Mô tả
moveSlide(step) Di chuyển slider theo một số bước (step) nhất định. Nếu step là số dương, slider sẽ di chuyển về phía trước. Nếu là số âm, nó sẽ di chuyển về phía sau.
